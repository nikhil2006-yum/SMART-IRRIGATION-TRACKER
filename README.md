# 🌱 Smart Irrigation System using Raspberry Pi Pico

A low-cost, sensor-based Smart Irrigation System designed for efficient agricultural water management using Raspberry Pi Pico (RP2040). This project automates irrigation by monitoring real-time soil moisture, temperature, and humidity while also providing approximate NPK estimation for smarter fertilizer management.

---

## 🚀 Features

* 🌾 Automated irrigation based on soil moisture levels
* 🌡️ Temperature & humidity monitoring using DHT11
* 💧 Servo-controlled water valve automation
* 📡 ESP32-CAM remote monitoring support
* 🧪 Approximate NPK estimation algorithm
* 🔋 Battery-powered portable design
* 📊 Real-time sensor data display
* 💸 Low-cost and scalable agricultural solution

---

## 🛠️ Hardware Components

* Raspberry Pi Pico (RP2040)
* Soil Moisture Sensor
* DHT11 Sensor
* Servo Motor
* Relay Module
* Water Pump
* ESP32-CAM Module
* Battery Pack
* Buck Converter
* AMS1117 Voltage Regulator

---

## 📈 Working Principle

1. Soil moisture and environmental data are continuously monitored.
2. Raspberry Pi Pico processes sensor inputs.
3. If moisture falls below threshold:

   * Servo valve opens
   * Water pump activates
4. Irrigation stops automatically after reaching desired moisture level.
5. NPK values are estimated for fertilizer guidance.
6. ESP32-CAM enables remote field monitoring.
